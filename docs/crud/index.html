Dile Components Crud System

The dile-components catalog offers the @dile/crud package, a complete set for creating advanced and fully configurable CRUD systems. Thanks to these components and utilities, you can build administration panels quickly and easily, avoiding tedious and repetitive tasks.

This CRUD system is designed to communicate via Ajax with a backend that exposes data and resource operations as a REST API. You can adjust everything from its appearance to the backend communications, so it adapts to the format of requests or responses. In addition, it provides a CLI to easily and quickly generate the scaffolding for components.

To learn more about the features of this library, we recommend starting with the CRUD general information page.
