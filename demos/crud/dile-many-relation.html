dile-many-relation component

Demo de gestión de la relación muchos-a-muchos entre Board Games y Tags. Selecciona un tag del desplegable para añadirlo al juego, o usa el botón × para eliminarlo.

dile-many-relation component - Add on Select

Demo con comportamiento más ágil usando addOnSelect. Con este modo, simplemente selecciona un tag del desplegable y se añade automáticamente a la lista, sin necesidad de hacer clic en un botón adicional. Usa el botón × para eliminar un tag.
