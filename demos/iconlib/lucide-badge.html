CSS Custom Properties

All CSS custom properties that can be customized:

  • --dile-lucide-badge-color: Background color (default: --dile-primary-color)
  • --dile-lucide-badge-on-color: Text color (default: --dile-on-primary-color)
  • --dile-lucide-badge-font-size: Text size (default: 0.8rem)
  • --dile-lucide-badge-font-weight: Text weight (default: 500)
  • --dile-lucide-badge-gap: Space between icon and text (default: 0.5rem)
  • --dile-lucide-badge-padding: Inner spacing (default: 0.375rem 0.75rem)
  • --dile-lucide-badge-border-radius: Border radius (default: 9999px for pill shape)
  • --dile-lucide-badge-border-width: Border thickness (default: 0px)
  • --dile-lucide-badge-border-color: Border color (default: transparent)
  • --dile-lucide-badge-transition-duration: Animation duration (default: 0.3s)
  • --dile-lucide-badge-icon-size: Icon size (default: 18px)

Variant-specific properties:

  • --dile-lucide-badge-primary / --dile-lucide-badge-on-primary
  • --dile-lucide-badge-secondary / --dile-lucide-badge-on-secondary
  • --dile-lucide-badge-success / --dile-lucide-badge-on-success
  • --dile-lucide-badge-warning / --dile-lucide-badge-on-warning
